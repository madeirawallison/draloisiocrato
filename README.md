# Página em construção - Dr. Aloísio Brasil

Arquivos prontos para subir no GitHub Pages.

## Como publicar
1. Crie um repositório no GitHub.
2. Envie o arquivo `index.html` e a pasta `assets` para a raiz do repositório.
3. Ative o GitHub Pages em Settings > Pages.
4. Aponte o domínio `www.draloisiocrato.com.br` para o GitHub Pages.
